# Nexpr

Nexpr is a data-driven running platform that helps athletes turn training data into clear, practical decisions. Connect Strava to track progress, spot trends, and train with better context.

> Independent app. Not affiliated with or endorsed by Strava.

## What Nexpr Does

- Predicts race potential and PR probability
- Tracks training momentum, freshness, and readiness trends
- Analyzes fueling habits alongside run outcomes
- Logs recurring mechanical stress signals per activity
- Supports goal planning with projected completion timelines
- Captures weekly reflections to connect training blocks over time

## Core Views

| View       | Purpose                                             |
| ---------- | --------------------------------------------------- |
| Dashboard  | High-level performance snapshot and recent activity |
| Activities | Full run history with context and filters           |
| Insights   | Fitness metrics, predictions, and trend analysis    |
| Goals      | Goal progress and forecasted outcomes               |
| Journal    | Weekly reflection and training notes                |
| Tools      | VDOT and marathon diagnostics utilities             |
